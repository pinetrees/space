Software Developer

This experience allowed me to inch my way into the development community, eventually learning and working with each of the languages listed here. I, like many developers, found each to have a character, and unlike many developers, did not have the freedom to decide which ones I used. With this said, I developed a very strong proficieny with PHP, Python, and Javascript, and was able to use this not only to solve many problems for existing applications, but also to write applications from the ground up for anyone with an idea.
